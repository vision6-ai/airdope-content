## Bring Your Existing Fans

If you already have a mailing list, phone number list, or contacts from another platform, you can import them into Airdope's CRM to start engaging them directly.

## What You Can Import

- Email addresses
- Phone numbers
- Names
- Custom tags (to preserve any existing segmentation)

## How to Import

### Step 1: Prepare Your File

Create a CSV file with your contacts. Supported columns:

| Column | Required | Example |
|--------|----------|---------|
| email | Yes (or phone) | fan@example.com |
| phone | Yes (or email) | +1234567890 |
| name | No | Jane Doe |
| tags | No | vip, nyc-show |

### Step 2: Upload

1. Go to **Fans CRM** → **Import**
2. Upload your CSV file
3. Map your columns to Airdope fields
4. Review the preview
5. Click **Import**

### Step 3: Invite to Wallet Pass

After importing, send an invitation:
- Email contacts receive an invite to add your wallet pass
- Phone contacts receive an SMS with your wallet pass link

## Important Notes

- **Consent required** — Only import contacts who have opted in to hear from you
- **Deduplication** — Airdope automatically skips contacts that already exist
- **Format** — Phone numbers should include country code (e.g., +1 for US)
- **Limits** — Free plan: up to 500 imports. Paid plans: unlimited.

## Migrating from Other Platforms

If you're coming from Mailchimp, ConvertKit, or another platform:
1. Export your contacts as CSV from the old platform
2. Follow the import steps above
3. Optionally tag them with the source (e.g., "mailchimp-import")
